# 作物种子模式识别

#### 介绍
这是我们小组合作完成的模式识别大作业项目——作物种子模式识别。
项目存放在码云上（暂时为私有）。

#### 软件架构
采用Python实现。


#### 使用说明
KNN分类算法	 
```
python judge.py
```
贝叶斯分类算法	 
```
python judge_bayes.py
```
层次聚类算法  
```
python hierarchical_clustering.py
```
K-Means聚类算法	 
```
python k_means.py
```
